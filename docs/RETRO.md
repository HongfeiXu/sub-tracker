## Phase 1 — 2026-02-26

- Vite 脚手架在非空目录下会取消操作，需要先在临时目录创建再复制过来
- Tailwind v4 的 `@custom-variant dark` 配置顺利，暗色模式手动 class 切换无问题
- 单文件方案下 TypeScript 的 `noUnusedLocals` 会报后续阶段才用到的常量，通过 export 解决
- Phase 1 整体顺利，无遗留问题
